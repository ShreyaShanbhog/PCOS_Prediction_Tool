# Work Breakdown Structure (WBS)

## 1. Data Collection
- Gather RNA-seq datasets.
- Collect ultrasound images.

## 2. Data Processing
- Gene: QC, normalization.
- Images: preprocessing.

## 3. Feature Extraction
- Gene: identify differentially expressed genes.
- Image: extract features using CNN.

## 4. Integration & Analysis
- Combine gene + image features.
- Dimensionality reduction (UMAP, PCA).
- Cluster samples and predict PCOS subtypes.

## 5. Visualization
- Heatmaps, plots, subtype distribution.

## 6. Reporting
- Generate result files and summary statistics.
